# shared/CLAUDE.md

Shared package guidance. See [/CLAUDE.md](../CLAUDE.md) for project overview, monorepo map, request flow, and global code style.

Imported by `web/` and `server/` as `fractapay-shared`. No commands, no env vars — pure TypeScript types/constants/helpers. Inherits global TypeScript code style from the root.

## Architecture

### Shared internals

- **Types** — `TToken = 'TESOURO'`; `TLanguage = 'en-US' | 'pt-BR'`; `TFiatCurrency = 'BRL'`; `TPixKeyType = 'EVP' | 'CPF' | 'CNPJ' | 'EMAIL' | 'PHONE'`; `TPayment = { id, amount, description? }`; `TPaymentResponse = { payments }`; `TUploadPayload = { file, token, address? }` (address optional — destination is selected separately); `TUploadResult = { success, payments, error? }`; `TRecipientShare = { address, percentage }`; `TDestination = { id, name, token, pixKey, pixKeyType }` (no `stellarAddress` — the global address comes from `usePaymentsStore`); `TDestinationAllocation = { destination: TDestination, percentage: number }`; `TChatMessage`, `TChatRequest`, `TChatHistoryMessage`, `TChatAction` (includes `update_payments` with delta `add | remove` per item); `TChatResponse` — no `price` field; `TChatPayment` — no `price` field; `TPaymentSummaryItem = { destinationName, token, amount, percentage, feeAmount?, totalAmount? }` (server computes `feeAmount` and `totalAmount` from `FEE_PERCENTAGE`; `token` derived from `allocation.destination.token`). **DB payment types**: `TPayment` (the persisted payment record) — includes `externalId: string` (non-nullable, renamed from `orderId`), `tokenAmount: string` (non-nullable), `feeAmount: string` (non-nullable — the actual fee charged, from the Etherfuse quote), `feePercentage: string` (non-nullable — the fee percentage applied at payment time, e.g. `"2"` for 2%). `TCreatePaymentPayload` — includes `feeAmount: string` and `feePercentage: string`. **Etherfuse types**: `TKycStatus`, `TOnboardingPayload`/`TOnboardingResult`, `TKycStatusResult`, `TBankAccountPayload`/`TBankAccountResult`, `TQuotePayload`/`TQuoteResult`, `TOrderPayload`/`TOrderResult` (with `pix?: TPaymentPix`), `TPaymentPix`, `TPaymentStatus = 'CREATED' | 'FUNDED' | 'COMPLETED' | 'FAILED' | 'REFUNDED' | 'CANCELED'`. **Auth types**: `TUser = { id, email, name: string | null, picture: string | null, onboardingCompletedAt: string | null }` (`picture` maps from the DB `avatarUrl`; `onboardingCompletedAt` is an ISO string or null); `TAuthMeResult = { success: true; user: TUser } | { success: false; error: EErrorCode }`; `TCompleteOnboardingPayload = { companyName }`; `TExchangePayload = { code, verifier }` and `TExchangeResult = { success: true; token } | { success: false; error: EErrorCode }` (the PKCE code→JWT exchange). `EErrorCode` enum adds: `INVALID_PAYLOAD`, `ETHERFUSE_REQUEST_FAILED`, `KYC_NOT_APPROVED`, `QUOTE_EXPIRED`, `ORDER_NOT_FOUND`, `PENDING_ORDER_EXISTS`, `CUSTOMER_ALREADY_EXISTS`, `CUSTOMER_NOT_FOUND`, `FILE_PARSE_FAILED`, `NO_PAYMENTS_FOUND`, `UNAUTHORIZED`, `SESSION_EXPIRED`, `OAUTH_FAILED`, `INVALID_AUTH_CODE`, `NOT_IMPLEMENTED`.
- **Constants** — `APP_NAME = 'FractaPay'`, `ALLOWED_EXTENSIONS`, `ALLOWED_MIME_TYPES`, `ALLOWED_INPUT_ACCEPT`, `SUPPORTED_TOKENS`, `TOKENS: TToken[]`, `PIX_KEY_TYPES: TPixKeyType[]`, `PAYMENT_MESSAGE_ROLES`, `PAYMENT_STATUSES`, `STELLAR_DECIMALS = 7`, `FIAT_BY_TOKEN = { TESOURO: 'BRL' }`, `LANGUAGE_BY_TOKEN`, `SYMBOL_BY_TOKEN = { TESOURO: 'R$' }`, `RECIPIENT_PERCENTAGE = BigNumber('0.15')`, `FEE_PERCENTAGE = BigNumber('0.02')`, `QUOTE_EXPIRY_SECONDS = 60`.
- **Helpers** — `ErrorHelper.map(error)` maps unknown errors to `EErrorCode`. `StellarHelper.isValidAddress(value)`. `StringHelper.truncateMiddle`, `StringHelper.formatAmount`, `StringHelper.formatCurrencyAmount`.
- **Helpers** — `StellarHelper.isValidAddress(value)` (wraps `StrKey.isValidEd25519PublicKey`). `StringHelper.truncateMiddle(value, max)`, `StringHelper.formatAmount(value)` (accepts `string | number | BigNumber`, Stellar 7-decimal cap with `ROUND_DOWN`), `StringHelper.formatCurrencyAmount(value, token)` (locale + symbol via `Intl.NumberFormat`, fixed at 2 decimals for display).
