# shared/CLAUDE.md

Shared package guidance. See [/CLAUDE.md](../CLAUDE.md) for project overview, monorepo map, request flow, and global code style.

Imported by `web/` and `server/` as `fractapay-shared`. No commands, no env vars — pure TypeScript types/constants/helpers. Inherits global TypeScript code style from the root.

## Architecture

### Shared internals

- **Core types** — `TToken = 'TESOURO'`; `TLanguage = 'en-US' | 'pt-BR'`; `TFiatCurrency = 'BRL'`; `TPixKeyType = 'evp' | 'cpf' | 'cnpj' | 'email' | 'phone'`; `TPayment = { id, amount, description? }` (`amount` is the FIAT/BRL value); `TPaymentResponse = { payments, price }`; `TDestination = { id, name, token, pixKey, pixKeyType }` (no per-destination Stellar address — the recipient address is global, taken from the authenticated user's smart-account `stellarAddress`); `TDestinationAllocation = { destination: TDestination, percentage: number }`; `TPaymentSummaryItem = { destinationName, token, amount, percentage, feeAmount?, totalAmount? }` (server computes `feeAmount`/`totalAmount` from `FEE_PERCENTAGE`; `token` derived from `allocation.destination.token`).
- **Chat types** — `TChatRole = 'user' | 'assistant'`; `TChatHistoryMessage = { role, content }`; `TChatMessage = { id, role, content, type: 'text' | 'file-import' | 'summary', payments?, summary?, timestamp }`; `TChatAction = 'none' | 'add_payments' | 'update_payments' | 'set_allocations' | 'request_confirmation' | 'execute' | 'clear'`; `TChatResponse = { message, action, payments?, price?, allocations?, summary?, errorCode?, filename? }`. (There is no `TChatRequest` — the `/chat` route reads `messages` + `context` as multipart fields.)
- **Auth types** — `TUser = { id, email, name: string | null, picture: string | null, onboardingCompletedAt: string | null, stellarAddress: string | null, passkeyCredentialId: string | null }` (`picture` maps from the DB `avatarUrl`; `stellarAddress` is the passkey smart-account contract id `C…`); `TAuthMeResult = { success; user } | { success: false; error }`; `TCompleteOnboardingPayload = { companyName, stellarAddress, passkeyCredentialId }`; `TExchangePayload = { code, verifier }` / `TExchangeResult = { success; token } | { success: false; error }` (PKCE code→JWT exchange).
- **Etherfuse types** — `TKycStatus = 'not_started' | 'pending' | 'approved' | 'rejected'`; `TOnboardingPayload = { publicKey }` / `TOnboardingResult = { customerId, bankAccountId, presignedUrl }`; `TKycStatusResult = { status }`; KYC identity submission — `TKycName`, `TKycAddress`, `TKycIdNumber`, `TKycIdentity`, `TSubmitKycPayload = { publicKey, identity }`, `TSubmitKycResult = { status, message }`; `TBankAccountPayload`/`TBankAccountResult`; `TQuotePayload = { customerId, sourceAmount, token, publicKey }` / `TQuoteResult = { quoteId, sourceAmount, destinationAmount, exchangeRate, feeAmount, etherfuseFeeAmount, fractapayFeeAmount, expiresAt, createdAt }`; `TOrderPayload = { quoteId, customerId, bankAccountId, publicKey, memo? }` / `TOrderResult = { orderId, status, pix?, confirmedTxSignature?, amountInFiat?, amountInTokens?, isRecovered? }`; `TPixInstructions`; `TOrderStatus = 'created' | 'funded' | 'completed' | 'failed' | 'refunded' | 'canceled'`.
- **Balance / organization types** — `TBalanceResult = { token, balance, balanceInFiat }` (on-chain TESOURO balance + its BRL value); `TOrganizationPayload = { displayName, accountType: 'personal' | 'business', email, userDisplayName, publicKey }`, `TOrganizationWallet = { id, publicKey, blockchain }`, `TOrganizationResult = { organizationId, displayName, accountType, wallets }`.
- **`ErrorCode` enum** — `NO_FILE`, `UNSUPPORTED_FILE_TYPE`, `INVALID_TOKEN`, `INVALID_ADDRESS`, `INVALID_PAYLOAD`, `AI_PARSE_FAILED`, `NETWORK_ERROR`, `RATE_FETCH_FAILED`, `ORDERBOOK_FETCH_FAILED`, `ETHERFUSE_REQUEST_FAILED`, `KYC_NOT_APPROVED`, `QUOTE_EXPIRED`, `ORDER_NOT_FOUND`, `PENDING_ORDER_EXISTS`, `CUSTOMER_ALREADY_EXISTS`, `CUSTOMER_NOT_FOUND`, `FILE_PARSE_FAILED`, `NO_PAYMENTS_FOUND`, `UNAUTHORIZED`, `SESSION_EXPIRED`, `OAUTH_FAILED`, `INVALID_AUTH_CODE`, `NOT_IMPLEMENTED`, `ONBOARDING_ALREADY_COMPLETED`, `BALANCE_FETCH_FAILED`, `UNKNOWN`.
- **Constants** — `ALLOWED_EXTENSIONS`, `ALLOWED_MIME_TYPES`, `ALLOWED_INPUT_ACCEPT`, `TOKEN = { TESOURO: 'TESOURO' }`, `SUPPORTED_TOKENS = [TESOURO]`, `SUPPORTED_LANGUAGES = ['en-US', 'pt-BR']`, `DEFAULT_LANGUAGE = 'en-US'`, `STELLAR_DECIMALS = 7`, `FIAT_BY_TOKEN = { TESOURO: 'BRL' }`, `LANGUAGE_BY_TOKEN = { TESOURO: 'pt-BR' }`, `SYMBOL_BY_TOKEN = { TESOURO: 'R$' }`, `RECIPIENT_PERCENTAGE = new BigNumber('0.15')`, `FEE_PERCENTAGE = new BigNumber('0.02')`, `QUOTE_EXPIRY_SECONDS = 60`.
- **Helpers** — `StellarHelper.isValidAddress(value)` (ed25519 `G…` via `StrKey.isValidEd25519PublicKey`), `StellarHelper.isValidContractAddress(value)` (contract `C…` via `StrKey.isValidContract`), `StellarHelper.isValidStellarDestination(value)` (accepts **either** `G…` or `C…` — used for the smart-account recipient, which is a contract address). `StringHelper.truncateMiddle(value, max)`, `StringHelper.formatAmount(value)` (accepts `string | number | BigNumber`, Stellar 7-decimal cap with `ROUND_DOWN`), `StringHelper.formatCurrencyAmount(value, token)` (locale + symbol via `Intl.NumberFormat`, fixed at 2 decimals for display).
